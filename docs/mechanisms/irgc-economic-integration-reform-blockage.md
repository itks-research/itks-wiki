# ادغام اقتصادی سپاه پاسداران → مسدود کردن اصلاحات

**دامنه:** ایران محور
**فرآیند:** L2.1
**منابع شواهد:** 0

## توصیف

وقتی سپاه پاسداران انقلاب اسلامی (IRGC) بر دارایی‌های اقتصادی مهم (بندرها، انرژی، تولید، ساخت و ساز) کنترل دارد، می‌تواند اصلاحات را که به منافع تجاری خود تهدید می‌کند، مسدود کند. این مکانیسم ایران محور توضیح می‌دهد که چگونه یک سازمان امنیتی به عنوان مانعی ساختاری به لیبرالیزاسیون تبدیل می‌شود: فرماندهان سپاه پاسداران خصوصی‌سازی را متوقف می‌کنند، سرمایه‌گذاری خارجی را مخالفند و از قدرت اقتصادی برای حفظ غلبه امنیت استفاده می‌کنند.
