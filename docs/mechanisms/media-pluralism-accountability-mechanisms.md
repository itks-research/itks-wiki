# Media pluralism → accountability mechanisms

**Domain:** Institutional
**Framework Node:** L3.4
**Evidence Sources:** 0

## Description

When media systems transition from state monopoly to competitive pluralism, multiple independent outlets create accountability mechanisms that constrain executive power and expose corruption. Investigative journalism, editorial debate, and competing narratives institutionalize transparency and make authoritarianism harder to reimpose without visible visible resistance.

## Context Requirements

- **Media Ownership Concentration:** distributed_across_factions
- **Regulatory Independence:** media_authority_insulated_from_politics
- **Journalist Safety:** legal_protections_and_enforcement
- **Audience Literacy:** sufficient_for_media_consumption
- **Advertising Markets:** competitive_enabling_independent_outlets

## Expected Outcomes

Expanded political accountability; exposure of elite wrongdoing; informed citizenry; institutionalized transparency

## Typical Triggers

- Broadcasting license system opens to new outlets
- First independent newspaper publishes investigation
- Private TV stations air opposition coverage
- Internet media platforms break through censorship
- Competitive journalism establishes reputational stakes

**Typical Timeframe:** 36 months
