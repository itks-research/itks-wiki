# راهنمای استفاده از سایت

<div dir="rtl" markdown>

این سایت یک سامانه دانش پژوهشی با بیش از ۵۰۰۰ منبع آکادمیک درباره مسیرهای توسعه ایران است. در اینجا نحوه استفاده موثر از آن را توضیح می‌دهیم.

## نقاط شروع

### یافته‌های پژوهشی
برای بررسی عمیق موضوعی از اینجا شروع کنید. هر حوزه (مانند طراحی قانون اساسی، عدالت انتقالی) دارای مقاله کلی و چندین مقاله تخصصی است که شواهد چندین کشور را ترکیب می‌کند.

### مطالعات موردی
بر اساس کشور کاوش کنید. هر یک از ۸ صفحه مطالعه موردی (ایران، کره جنوبی، اسپانیا، لهستان، اندونزی، تونس، جمهوری چک، شیلی) منابع را بر اساس دسته‌بندی پژوهشی نشان می‌دهد.

### کاوشگر منابع
از [کاوشگر منابع](../explorer.md) برای یافتن مقالات خاص استفاده کنید:

- **جستجو** بر اساس عنوان، نویسنده یا کلیدواژه
- **فیلتر** بر اساس مطالعه موردی، دسته‌بندی، امتیاز ارتباط یا اعتبار
- **مرتب‌سازی** بر اساس هر ستون (سال، استنادها، ارتباط)
- **کلیک روی هر ردیف** برای مشاهده چکیده کامل، لینک DOI و فراداده
- **اشتراک‌گذاری** — آدرس URL با تغییر فیلترها به‌روز می‌شود

### نمودار دانش
[نمودار دانش](../graph.md) روابط بین منابع را تجسم می‌کند. گره‌ها نشان‌دهنده مقالات و یال‌ها نوع رابطه را نشان می‌دهند (تأیید، تناقض، گسترش، روش‌شناسی مشترک).

## درک امتیازها

- **امتیاز ارتباط (۱-۵):** میزان ارتباط منبع با مطالعه موردی و دسته‌بندی. ۵ = منبع اصلی، ۱ = ارتباط حاشیه‌ای
- **امتیاز اعتبار (۱-۵):** ارزیابی کیفیت بر اساس اعتبار نشریه، تعداد استنادها و داوری همتایان
- **تعداد استنادها:** تعداد دفعات استناد به مقاله توسط آثار آکادمیک دیگر

## درباره داده‌ها

تمام منابع از پایگاه‌های داده آکادمیک فهرست‌بندی شده‌اند. سامانه از فیلتر کیفیت سه‌مرحله‌ای استفاده می‌کند:

۱. **مرحله ۱:** امتیازدهی ارتباط
۲. **مرحله ۲:** ارزیابی اعتبار
۳. **ترکیب:** منابع با کیفیت بالا در مقالات تحلیلی ترکیب می‌شوند

سامانه دوزبانه است (انگلیسی و فارسی). از دکمه تغییر زبان برای جابجایی استفاده کنید.
