# How to Use This Site

This site is a research knowledge system with over 5,000 academic sources organized around Iran's development pathways. Here's how to navigate it effectively.

## Starting Points

### Research Findings
Start here for thematic deep dives. Each theme (e.g., Constitutional Design, Transitional Justice) has an overview article and several focused sub-articles that synthesize evidence from multiple countries.

### Case Studies
Explore by country. Each of the 8 case study pages (Iran, South Korea, Spain, Poland, Indonesia, Tunisia, Czech Republic, Chile) shows sources organized by research category, with links to the most relevant papers.

### Source Explorer
Use the [Source Explorer](../explorer.md) when you want to find specific papers. You can:

- **Search** by title, author, or keyword
- **Filter** by case study, category, relevance score, or reliability score
- **Sort** by any column (year, citations, relevance)
- **Click any row** to expand and see the full abstract, DOI link, and metadata
- **Share filtered views** — the URL updates as you filter, so you can share a link to a specific set of results

### Knowledge Graph
The [Knowledge Graph](../graph.md) visualizes relationships between sources. Nodes represent papers, edges show how they relate (supports, contradicts, extends, shared methodology). You can:

- Load different case studies from the dropdown
- Filter by category or case study
- Click nodes for details
- Toggle labels and clustering

## Understanding Scores

- **Relevance Score (1-5):** How relevant a source is to its assigned case study and category. 5 = core source, 1 = tangentially related.
- **Reliability Score (1-5):** Quality assessment based on venue reputation, citation count, methodology, and peer review status.
- **Citation Count:** Number of times the paper has been cited by other academic works.

## About the Data

All sources are indexed from academic databases (Semantic Scholar, OpenAlex, and others). The system uses a three-gate quality filter:

1. **Gate 1:** Relevance scoring — sources must meet minimum relevance thresholds
2. **Gate 2:** Reliability assessment — sources are evaluated for academic quality
3. **Synthesis:** High-quality sources are synthesized into thematic articles

The system is bilingual (English and Farsi). Use the language switcher to toggle between versions.
